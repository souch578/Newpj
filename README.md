A product details scraping code

```
Beautiful Soup , Requests , CSV
```

Items to scrape
• Product URL
• Product Name
• Product Price
• Rating
• Number of review
• Description
• ASIN
• Product Description
• Manufacturer 
The entire has been exported in a csv format : "product.csv"
